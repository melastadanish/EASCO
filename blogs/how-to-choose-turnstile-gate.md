### SECTION 1 — Intro / Hook

**H1: How to Choose a Turnstile Gate**

The right turnstile gate depends on three things. How many people pass through per minute, how much budget is available per entrance, and which credential type — card, biometric, or QR — needs to be supported. This guide breaks down the main gate types and the questions worth asking before requesting a quote.

### SECTION 2 — Start With Traffic Volume, Not Preference

**H2: Match the Gate to the Entrance, Not the Catalogue Page**

Most buying mistakes start with picking a gate style first and fitting it to the entrance afterward. Traffic volume should decide the category before anything else. A 300-seat auditorium exit needs a gate processing 45 or more people per minute — a cylindrical or speed gate range. A staff-only side door rarely needs more than a tripod turnstile, which costs less and takes up less floor space.

Once throughput is settled, credential type narrows the choice further. A gate needing to read face, card, and QR code from one terminal points toward an Android-integrated model. A gate needing only card access can use a simpler, lower-cost base unit.

> Choosing a turnstile gate starts with throughput — a cylindrical or speed gate handles 45–50 persons/min for high-traffic entrances, while a tripod turnstile suits lower-traffic, budget-conscious entrances — then narrows by credential type and environment (indoor vs. outdoor).

### SECTION 3 — Four Questions to Ask Before Requesting a Quote

**H2: What a Supplier Should Be Able to Answer Immediately**

1. **What's the confirmed throughput under normal conditions?** Not a marketing number — ask for the persons-per-minute figure the manufacturer will stand behind in writing.
2. **Is the anti-crush mechanism mechanical or software-only?** A mechanical stop (like crank-link transmission with magnetic-sensitive limit control) keeps working even if the control software fails.
3. **What happens during a power outage?** Confirm whether the gate fails open (for evacuation safety) or fails locked (for security), and whether that's configurable.
4. **What's the gate body's IP rating, separate from any attached terminal?** A biometric terminal's IP65 rating doesn't automatically mean the gate body itself carries the same rating.

### SECTION 6 — FAQ

**H2: Frequently Asked Questions**

**Q: How do I know which turnstile type fits my entrance?**
A: Start with traffic volume — how many people need to pass per minute at peak times. That alone eliminates most options before budget or credential type comes into play.

**Q: Is a more expensive gate always the better choice?**
A: No. A premium speed gate makes sense for high-traffic lobbies, but a tripod turnstile can be the better choice for a low-traffic staff entrance where the extra throughput and cost aren't needed.

**Q: What questions should I ask a turnstile supplier before ordering?**
A: Confirmed throughput, whether the anti-crush mechanism is mechanical or software-only, power-outage behavior, and the gate body's own IP rating separate from any attached terminal.

**Q: Can one gate handle face, card, and QR code access?**
A: Yes. Android-integrated gates run an onboard terminal supporting multiple credential types without a separate controller box.

**Q: Does a faster gate mean less safety?**
A: Not necessarily. A well-designed high-speed gate still includes anti-crush and anti-tailgating sensors — confirm both regardless of the gate's throughput rating.
