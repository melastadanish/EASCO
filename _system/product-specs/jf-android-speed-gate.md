## Key Selling Points

- Built-in Android system — all settings shown on screen, no external controller
- Convenient gate setup through 7-inch touchscreen
- Two positions to place the face recognition camera (flexible installation)
- Optional palm recognition camera
- 5 access methods in one gate: face, card, QR code, password, remote door opening
- SDK/API development supported
- Palmprint recognition option available

---

## Access Methods

1. Face recognition
2. QR code
3. Card reading (RFID/IC)
4. Password
5. Palmprint recognition (optional)
6. Remote door opening

## Configuration Relationship

- **J series** = Base aluminum speed gate (no Android terminal)
- **JF series** = J series + Android terminal = JF model
- **ESBL** = The embedded terminal that upgrades J → JF
