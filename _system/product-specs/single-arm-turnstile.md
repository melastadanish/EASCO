## Safety Features

- Anti-tailing: infrared sensor array
- Anti-pinch: automatic
- Automatic reset
- Automatic fault detection
- Voice broadcast
- Power-off gate open (fail-safe)
- RGB ambient light — Blue (standby) / Green (pass) / Red (denied/abnormal)
