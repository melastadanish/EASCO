## Key Selling Points

- Flowing water / iridescent light effect provides better passing direction guidance
- Aluminum material — customizable with logo on gate body
- Brand-new direct-drive servo motor — quieter, longer lifespan
- < 40dB noise level
- Absolute encoder technology for precise control
- MCBF > 30,000,000 cycles (motor lifespan)
- Zero screws visible on gate body
- Optional Android terminal — face, card, QR code, password, remote access
