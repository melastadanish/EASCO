## 一、简介
该产品工作电压处于36 伏以下的人体安全电压范围，即便被水浸泡，也
绝不导电触电，能切实保障使用安全。
齿条平移门开门机控制驱动器由一个减速正弦电机+控制器组成的机电一
体化产品，采用了悬浮限位运行，保证门直线运行，安装了此控制驱动器的悬浮
门机头在安装接入了极限位置开关、红外防撞传感器、地感信号、红外信号、遥
控和手控开关等信号后，可以控制驱动伸缩门按照控制信号的要求进行相应的动
作，从而实现对于出入口的车辆和行人的管控。
减速正弦电机驱动控制器的主要特点是：启动力矩大，温升极小，控制精
准，运行平稳、噪声小、安全等特点，是悬浮门厂家理想的控制驱动器。
该产品不用担心线细产生压降，电机安装底座可无缝替换交流电机产品。

## 三、主要功能性能说明及产品特点
由于采用限位运行，保证平移门安装设定的直线运行。
3.1 主要功能说明
1. 防撞功能：本控制器可以根据需求接入防撞光电探头，当门体关闭过程中前
方由障碍物时，门体自动停止运行；
2. 遇阻功能：当门体在关闭过程中遇到阻力且达到设定电流值和时间后，则停
止运行；
3. 半开门功能：当线控信号接到半开门端子后，可以将门打开设定的宽度（非
全开），然后收到关门信号后或者地感检测到车辆过后关闭（该功能主要时配
合自动控制车辆出入使用）；
4. 遥控功能：驱动控制器标配遥控功能，可以通过遥控器控制门的“开”“关”
“停”；

5. 指令控制功能：控制器可以通过串口和上位机通讯，实现指令控制门的运行；
6. 一键恢复功能：可以实现一键恢复出厂设置；
7. 开机和运行自检功能：控制可以检测设备的运行状态并及时告警；
8. 防冻功能：控制驱动器可以进行防冻处理，保证设备在较低环境温度下运行。
9. 拥有微信小程序远程控制开关门（可了解门的状态），可统计门的使用累计
次数及当日使用次数，并支持远程调试功能，还能主动上报各类故障，采用
了先进的伺服控制算法。
10. 可通过微信小程序设置时段开门、长时间开门，也可以设置时段禁止开门。
### 3.2 主要性能说明
1.具备3 倍过载能力，且支持缓起缓停功能。
2. 电机不存在热保护问题，能够实现多级调速。
### 3.3 产品主要特点
低压直流36 伏的伺服齿条平移门开门机控制系统具备一系列显著特点，使
其在安全性、功能性和便捷性等方面表现出色。
1. 低压直流36 伏，被水浸泡无导电触电风险，保障人身安全。
2. 3 倍过载能力及缓起缓停功能。
3. 微信小程序实现远程控制、故障上报、伺服控制算法应用，开闸时段设置。
4. 无电机热保护问题。
5. 具备多级调速功能。
